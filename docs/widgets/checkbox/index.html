Checkbox 插件说明
  • id

    这是有 Meetbot 自动生成的编号，你需要将带有此编号的元素放入您的网页中，SDK 将会自动将其按照配置渲染成为对应的插件。

  • type

    这是一个固定的值Checkbox，表示此插件的类型。

  • pageId

    Facebook 页面 ID，表示您的 Bot 所连接的 Facebook 页面的编号。

  • origin

    插件加载网址的基域，必填。

    指示插件在哪个网站加载，如果不填该项，插件将会无法显示。

    在这里我们将其写死成当前网页，作为展示用。

  • skin

    插件内容的色彩主题，选填。

    浅色主题插件显示深黑色文本

    深色主题插件显示白色文本、透明 messenger 图标和白色闪电图标

    默认为浅色主题。

  • size

    插件大小，表示插件呈现的尺寸大小，选填。

    默认为大。

  • allowLogin

    让用户能够在没有现有会话的情况下登录，同时启用“不是你”选项，选填。

    默认为true。

  • centerAlign

    插件内容是否居中对齐，选填。

    默认为false。

  • hideAfterChecked

    用户勾选确认后多少天内自动隐藏，选填。

    默认不启用此功能。

    • 预览
    插件加载网址的基域 色彩主题 (选填) 插件大小 (选填) 是否允许用户登录 (选填) 是否居中对齐 (选填) 选中后隐藏 (选填)
