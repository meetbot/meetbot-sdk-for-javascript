Message Us 插件说明和预览
  • id

    这是有 Meetbot 自动生成的编号，你需要将带有此编号的元素放入您的网页中，SDK 将会自动将其按照配置渲染成为对应的插件。

  • type

    这是一个固定的值MessageUs，表示此插件的类型。

  • pageId

    Facebook 页面 ID，表示您的 Bot 所连接的 Facebook 页面的编号。

  • color

    主题颜色，表示插件的主要背景色，选填。

    默认为blue。

  • size

    插件大小，表示插件呈现的尺寸大小，选填。

    默认为large。
