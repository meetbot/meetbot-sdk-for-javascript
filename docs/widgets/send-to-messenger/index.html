Send To Messenger 插件说明和预览
  • id

    这是有 Meetbot 自动生成的编号，你需要将带有此编号的元素放入您的网页中，SDK 将会自动将其按照配置渲染成为对应的插件。

  • type

    这是一个固定的值SendToMessenger，表示此插件的类型。

  • pageId

    Facebook 页面 ID，表示您的 Bot 所连接的 Facebook 页面的编号。

  • color

    主题颜色，表示插件的主要背景色，选填。

    默认为blue。

  • size

    插件大小，表示插件呈现的尺寸大小，选填。

    默认为large。

  • enforceLogin

    表示用户是否需要强制登录，选填。

    如果为true，则点击该按钮时，即便是已登录用户也必须重新登录。

    默认为false

  • ctaText

    按钮上显示的文本，选填。

    默认为Send to Messenger。

    • 预览
    主题颜色 (选填) 插件大小 (选填) 是否强制登录 按钮文本
